The :mod:`macaron` module is the main module of *Macaron*.
It has some module functions which are needed to use *Macaron*.


Usage
=====

*Macaron* works with a global :class:`Macaron` object.
The object is generated by the :mod:`macaron` module.
Typical usage starts with :func:`macaronage`
and finishes with :func:`cleanup`.
If you change the database contents,
you will need to call :func:`bake` (like commit)
or :func:`rollback`.

Using database models is described in :doc:`class/model`.

Example
-------

::

    import macaron
    
    # Initialize with autocommit and SQL execution logging
    macaron.macaronage("mybook.db", autocommit=True, history=10)
    
    ... uses Models ...
    
    # print out the last five history entries
    for idx in range(0, 5):
        print macaron.history[idx]
    
    # finish macaron
    macaron.bake()
    macaron.cleanup()


Module attributes
=================

.. attribute:: macaron.history

   :exception: RuntimeError -- SQL history is not enabled.
   :exception: IndexError -- index is lager than history count.
   :rtype: string or list

   Returns history of SQL execution. 
   You can get history like a list (index:0 is latest).
   It may be useful for debugging your applications.

   ::

      last_sql = macaron.history[0]
      second_last_sql = macaron.history[1]
      
      sqls = macaron.history[0:5]

Logging
=======

SQL logging
-----------

*Macaron*'s logging architecture uses :mod:`logging`
which is a Python standard module.
As a simple example, you can use SQL history
by calling :func:`macaron.macaronage` with the ``history`` parameter.
It sets up a logger for recording executions of SQL,
and you can obtain the SQL and its stringfied parameters
through :attr:`macaron.history`.

Using a custom logger
-------------------

If you want to use custom logger(s),
you call :func:`macaron.macaronage` with the ``logger`` parameter.
It sets a logger for recording history, too.
The class of the logger object must be a subclass of :class`logging.Logger`.

If you set both ``logger`` and ``history`` parameters,
*Macaron* will add a handler to your logger object.
